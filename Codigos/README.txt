# Sistema de Compras

Um sistema de compras desenvolvido em Python para gerenciar os pedidos dos fornecedores.

## Descrição

Este projeto é uma aplicação para gerenciar o processo de compras em uma empresa. Ele permite que os usuários registrem pedidos, gerenciem fornecedores e acompanhem o status das compras. O sistema foi desenvolvido utilizando Python e PyMySQL para a conexão com o banco de dados MySQL.


### Pré-requisitos

- Python 3
- MySQL
- PyMySQL
- pip (gerenciador de pacotes do Python)
- Arquivo sendo localizado no (C:)
- Banco utilizado no sv: 192.162.1.19
